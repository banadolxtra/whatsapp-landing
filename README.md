# لينكد يمن

صفحة هبوط عربية responsive لبيع الاشتراكات الرقمية، مع أزرار CTA تفتح واتساب على الرقم `967738544822` برسالة تلقائية حسب المنتج.

## النشر على GitHub Pages

1. ارفع الملفات إلى مستودع GitHub عام.
2. من إعدادات المستودع افتح **Settings > Pages**.
3. اختر **Deploy from a branch**.
4. اختر فرع `main` والمجلد `/root`.
5. احفظ الإعدادات وانتظر رابط النشر.

## تعديل رسائل واتساب

عدّل قيمة `data-message` داخل أزرار المنتجات في `index.html`.
